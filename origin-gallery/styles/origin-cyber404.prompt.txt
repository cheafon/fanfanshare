请生成一个单页 HTML 网页，要求如下：

【网页类型】404 错误页面 / Page Not Found
【主题】一个有趣的、沉浸式的 404 页面，让用户在迷路时也能感到惊喜
【品牌】通用 — 适用于任何科技/创意公司

【视觉风格】赛博朋克 Cyberpunk / 霓虹雨夜
- 色调：深黑蓝 (#0a0e1a)、霓虹青 (#00f0ff)、霓虹品红 (#ff0080)、暗紫 (#1a0a2e)、钢灰 (#2a2a3a)
- 字体：Orbitron（方形未来数字体，英文标题）、Noto Sans SC 700/900（中文 17px+）
- 布局：全屏居中，巨大的 "404" 数字占据视觉焦点，故障效果（glitch），雨滴粒子背景，底部有返回按钮
- 动效：高密度 — CSS glitch 文字抖动（rgb 色差偏移）、雨滴粒子下落动画、霓虹闪烁脉冲、扫描线覆盖层
- 文化语境：赛博朋克/银翼杀手/攻壳机动队/霓虹雨夜

【页面结构】（404 错误页，不是商业模板）

1. 全屏背景：
   - 纯黑蓝底 + CSS 雨滴粒子（竖线从上到下）
   - CRT 扫描线覆盖层
   - 底部霓虹光晕（radial-gradient 模拟城市灯光倒影）

2. 主体居中内容：
   - "404" 超巨大数字（15rem+，Orbitron，霓虹青色）
   - CSS Glitch 效果：文字带 rgb 色差偏移动画（::before 品红偏移 ::after 青色偏移）
   - "页面未找到" 中文副标题
   - "你来到了一个不存在的维度" 有趣的描述文字
   - "ERROR: DIMENSION_NOT_FOUND" 等宽小字错误码

3. 交互区：
   - "返回首页" 按钮（霓虹边框发光）
   - "随机传送" 按钮（品红色，hover 发光）
   - 底部小字："如果你一直看到这个页面，可能是宇宙在暗示你该休息了"

4. 装饰元素：
   - 底部城市天际线剪影（CSS 几何方块模拟建筑）
   - 扫描线覆盖（repeating-linear-gradient）
   - 噪点纹理

【技术要求】
- 单个 HTML 文件，所有 CSS 和 JS 内联
- 使用 Google Fonts：Orbitron、Noto Sans SC
- 响应式设计，适配手机和桌面
- 中文为主，正文字号至少 17px
- 生产级品质
- 赛博朋克：glitch 色差动画、霓虹发光 text-shadow/box-shadow、雨滴粒子、扫描线、CRT 质感
