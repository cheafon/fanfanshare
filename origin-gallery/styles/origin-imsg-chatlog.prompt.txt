请生成一个单页 HTML 网页，要求如下：

【网页类型】聊天记录展示页 / Chat Log Showcase / 对话记录
【品牌】虚构展示：一段朋友间关于"今晚吃什么"的日常聊天记录
【主题】一个iOS iMessage风格的聊天记录展示页面，用真实的聊天气泡还原一段有趣的对话

【视觉风格】iMessage Glassmorphism / iOS毛玻璃聊天 / 苹果系统风
- 色调：iOS浅灰背景 (#f2f2f7)、iMessage蓝 (#007aff)、对方灰 (#e5e5ea)、文字黑 (#1c1c1e)、白色 (#ffffff)、绿色已读 (#34c759)、毛玻璃白 (rgba(255,255,255,0.72))
- 字体：SF Pro风格用 -apple-system 或 Nunito（圆润接近SF）英文、Noto Sans SC 400（中文 17px+）
- 布局：聊天界面式 — 模拟iPhone聊天界面，顶部毛玻璃导航栏（头像+名字+在线状态），中间聊天气泡区域（左灰右蓝，圆角气泡+小三角），底部输入栏
- 动效：高！
  * JS逐条消息staggered入场（每条消息延迟出现，模拟实时聊天）
  * CSS打字指示器（三个圆点交替跳动，在下一条消息出现前显示）
  * 气泡弹入动画（scale从0.8弹到1+translateY）
  * 毛玻璃顶栏（backdrop-filter:blur）
  * 时间戳渐显

【核心视觉技法 — 必须全部实现！】
1. 聊天气泡：左侧灰色圆角气泡（对方）+右侧蓝色圆角气泡（我），用border-radius的不对称值模拟iOS气泡形状，伪元素画小三角尾巴
2. 打字指示器：三个小圆点在灰色气泡内交替上下跳动（animation-delay错开），模拟"对方正在输入"
3. JS消息队列：页面加载后，消息一条一条弹入（每条间隔1-2秒），在消息出现前先显示打字指示器
4. 毛玻璃导航栏：顶部fixed，backdrop-filter:blur(20px)，半透明白色
5. 气泡弹入：每条消息从scale(0.85)和translateY(10px)弹入到正常
6. 在线状态：绿色小圆点+脉动动画

【页面结构】（聊天记录，像截图了一段iPhone聊天）

1. 顶部导航栏（毛玻璃fixed）：
   - ← 返回
   - 圆形头像占位（渐变圆形）
   - "小鱼" 名字
   - "在线" 绿色状态点+文字

   - 输入框占位："输入消息…"

【技术要求】
- 单个 HTML 文件，所有 CSS 和 JS 内联
- 使用 Google Fonts：Nunito（接近SF Pro圆润感）、Noto Sans SC
- 响应式设计，适配手机和桌面（最大宽度模拟手机屏幕）
- 中文为主，正文字号至少 17px
- 生产级品质
- 必须实现的特效：
  1. JS消息队列（14条消息逐条弹入，间隔1-1.5秒，打字指示器在每条前显示0.8秒）
  2. CSS打字指示器（3个圆点交替bounce）
  3. 气泡弹入动画（scale+translateY spring效果）
  4. 毛玻璃导航栏（backdrop-filter:blur(20px)）
  5. iOS风格气泡（不对称border-radius+伪元素三角尾巴）
  6. 在线状态脉动绿点
