请生成一个单页 HTML 网页，要求如下：

【网页类型】emoji猜词游戏 / Emoji Guessing Game / 看emoji猜成语
【品牌】虚构游戏「猜猜看 GUESS IT」— 看emoji猜成语小游戏
【主题】一个弹跳玩具风格的emoji猜成语游戏，用户根据emoji提示猜出对应成语

【视觉风格】Playful Bounce / 弹跳玩具 / 游戏化UI
- 色调：活力黄 (#FFD23F)、明亮蓝 (#4088FF)、快乐粉 (#FF6B9D)、活力绿 (#34D399)、纯白 (#FFFFFF)、深色文字 (#1A1A2E)、浅背景 (#F8F7FF)
- 字体：Fredoka（圆润粗体显示）英文、ZCOOL KuaiLe（中文圆润 17px+）、Noto Sans SC 400（正文）
- 布局：游戏UI式 — 浅色背景，顶部积分条，中央大emoji展示区，下方4个选项按钮（2×2网格），底部进度
- 动效：高！
  * emoji弹入动画（bounceIn scale）
  * 选项按钮hover弹跳
  * 正确：绿色边框+scale放大+checkmark
  * 错误：红色抖动shake
  * 下一题过渡：旧题滑出+新题滑入
  * 积分数字跳动

【页面结构】

1. 顶部积分条：
   - "猜猜看" logo
   - 分数：0分（JS更新）
   - 进度：第1题/共8题

2. 出题区（大emoji显示）：
   - 3-4个大emoji（如 🐴🏃💨 = 马不停蹄）
   - "这些emoji代表什么成语？"

3. 选项区（4个按钮，2×2网格）：
   - A. 正确答案
   - B. 干扰项
   - C. 干扰项
   - D. 干扰项
   - 每题选项随机排列

4. 8道题目库：
   - 🐴🏃💨 → 马不停蹄 (干扰：走马观花/一马当先/万马奔腾)
   - 👀🐑 → 一目了然... 不对，换：🐑🐑🐑🐑🐑 → 不可思议... 还是用经典的：
   - 🌧️🌧️☔🌧️ → 倾盆大雨 (干扰：暴风骤雨/风雨交加/大雨滂沱)
   - 🐔🐔🐶🐶 → 鸡犬不宁 (干扰：鸡飞蛋打/鸡犬升天/鸡鸣狗盗)
   - 🌸🌸🌸🌸 → 花好月圆... 用：💧💧🪨🪨 → 水落石出 (干扰：水到渠成/滴水穿石/石破天惊)
   - 🐯🐯🔥🔥 → 如虎添翼... 用：🔥🔥👀👀 → 火眼金睛 (干扰：怒火中烧/炉火纯青/星火燎原)
   - 🌙✨👀 → 望梅止渴... 用：🌍🌍🌊🌊 → 翻天覆地 (干扰：天翻地覆/开天辟地/惊天动地)
   - 🐟🐟💧💧 → 如鱼得水 (干扰：鱼目混珠/浑水摸鱼/缘木求鱼)
   - 🏔️🏔️🌊🌊 → 山高水长 (干扰：山清水秀/高山流水/千山万水)

5. 结果反馈：
   - 正确：绿色高亮+✓+弹跳
   - 错误：红色抖动+✗+显示正确答案

6. 结束画面（答完8题后）：
   - "游戏结束！"
   - 你的得分：X/8
   - 评价文字（根据分数）
   - "再来一局"按钮

7. 页脚：
   - "猜猜看 · GUESS IT"
   - "页面来源：虚构游戏 · 仅供展示"

【技术要求】
- 单个 HTML 文件，所有 CSS 和 JS 内联
- 使用 Google Fonts：Fredoka、ZCOOL KuaiLe、Noto Sans SC
- 响应式设计，适配手机和桌面
- 中文为主，正文字号至少 17px
- 生产级品质
- JS功能（完全可玩！）：
  1. 8题随机顺序出题
  2. 4选项每题随机排列
  3. 点击判断正误+反馈动画
  4. 积分累计
  5. 游戏结束统计+评价
  6. 重新开始
