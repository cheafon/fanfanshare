请生成一个单页 HTML 网页，要求如下：

【网页类型】番茄钟计时器 / Pomodoro Timer / 专注力工具
【品牌】虚构工具「静刻 STILL MOMENT」— 一个有温度的番茄钟
【主题】一个北欧温暖极简风格的番茄钟页面，帮助用户专注25分钟

【视觉风格】Scandinavian Hygge / 北欧温暖极简 / 功能之美
- 色调：暖白 (#FDFBF7)、温暖木色 (#C4956A)、深木 (#8B6847)、柔和灰 (#B8B0A8)、专注红 (#D4654A)、休息绿 (#6BA87A)、文字深 (#3A3530)
- 字体：Jost（优雅现代几何无衬线，北欧设计感）英文/数字、Noto Sans SC 300/500（中文 17px+）
- 布局：北欧极简式 — 暖白大面积留白，中央一个大圆形计时器（唯一视觉焦点），下方极简控制按钮，一切克制温暖
- 动效：低中（克制但精致）—
  * CSS圆形倒计时环（conic-gradient每秒更新）
  * 状态切换时颜色柔和过渡（专注红→休息绿）
  * 完成时轻微震动动画
  * 呼吸提示圆（开始前缓慢scale呼吸）
  * 数字变化柔和渐变

【页面结构】

1. 头部（极简）：
   - "静刻" 小字品牌名
   - "STILL MOMENT"

2. 圆形计时器（核心视觉，居中大圆）：
   - 外圈：conic-gradient进度环（专注时红色，休息时绿色）
   - 中心：大数字倒计时 "25:00"（Jost超大字）
   - 状态文字："专注时间" 或 "休息时间"
   - 当前轮次："第 1 轮 / 4"

3. 控制按钮（极简圆形按钮）：
   - ▶ 开始 / ⏸ 暂停（主按钮，大圆）
   - ⟲ 重置（小按钮）
   - 完成后自动切换到休息模式（5分钟）

4. 今日统计（底部小字）：
   - 已完成：0 个番茄
   - 今日专注：0 分钟

5. 设置提示：
   - "25分钟专注 + 5分钟休息 · 每4轮长休15分钟"

6. 页脚：
   - "静刻 · 把时间还给自己"
   - "页面来源：虚构工具 · 仅供展示"

【技术要求】
- 单个 HTML 文件，所有 CSS 和 JS 内联
- 使用 Google Fonts：Jost、Noto Sans SC
- 响应式设计，适配手机和桌面
- 中文为主，正文字号至少 17px
- 生产级品质
- JS功能（完全可用！）：
  1. 25分钟倒计时（每秒更新）
  2. 开始/暂停/重置控制
  3. 倒计时结束→自动切换休息模式(5分钟)
  4. CSS conic-gradient进度环实时更新
  5. 轮次计数（4轮一循环）
  6. 今日统计（完成番茄数+总专注分钟）
- CSS：
  1. 圆形conic-gradient进度环
  2. 颜色柔和过渡（红↔绿）
  3. 完成震动动画
  4. 开始前呼吸scale动画
