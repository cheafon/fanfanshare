【品牌】虚构App「觅食雷达 FOOD RADAR」— 发现你附近500米的好吃的
【主题】一个雷达扫描风格的附近美食推荐页面，美食点像雷达上的目标点

【视觉风格】Radar Scope / 雷达扫描 / 军事通信
- 色调：雷达深绿 (#0A1A0A)、磷光绿 (#30E848)、扫描线亮绿 (#60FF80)、暗绿 (#1A3A1A)、目标红 (#FF4040)、距离灰 (#2A3A2A)

- 布局：雷达屏幕式 — 深绿底，中央CSS雷达圆形（同心圆+旋转扫描线），美食目标点散布在雷达上，底部信息面板
- 动效：高！
  * CSS雷达扫描线旋转（conic-gradient扇形+持续旋转）
  * CSS同心圆距离环（border圆圈）
  * JS目标点随机闪烁（模拟雷达回波）
  * 扫描线扫过目标时亮起
  * 目标信息hover弹出

【页面结构】

1. 头部（军事通信风格）：
   - "觅食雷达" 标题（磷光绿）
   - "FOOD RADAR · 探测中…"
   - 当前位置：上海市静安区

   - 旋转扫描线（conic-gradient扇形，3s一圈）
   - 5个美食目标点散布在不同位置和距离

3. 美食目标（5个点位，在雷达上标记）：

4. 目标列表（底部信息面板，雷达数据风格）：
   - 每个目标一行：距离+名称+评分+类型

- 单个 HTML 文件，所有 CSS 和 JS 内联
- 使用 Google Fonts：VT323、Noto Sans SC
- 响应式设计，适配手机和桌面
- 中文为主，正文字号至少 17px
- 生产级品质

  1. CSS雷达扫描线(conic-gradient扇形渐变+3s旋转animation)
