【网页类型】互动趣味测试 / 性格测验页面
【品牌】星球研究所 PLANETLAB — 一个做趣味心理测试的内容平台
【测试】"你是哪种咖啡？" — 用5道选择题测试你的性格，最后告诉你是哪种咖啡
【主题】一个可以真正互动的趣味测试页，有选择、有结果、有分享

【视觉风格】Y2K 千禧未来 / 2000年代互联网美学
- 色调：钢蓝 (#4a8cc8)、液态银 (#c0c8d0)、千禧粉 (#ffa0c8)、全息紫 (#a060ff)、闪亮白 (#f0f0ff)、深蓝黑底 (#0a0a1a)
- 字体：Audiowide（未来几何宽体，英文标题）、ZCOOL QingKe HuangYou 站酷庆科黄油体（中文标题）、Noto Sans SC 17px+（正文）
- 布局：卡片式分步问答 — 每次显示一道题，选择后滑到下一题，最后显示结果。Chrome/金属质感边框，气泡/星星装饰
- 动效：高 — 卡片切换动画、选项 hover 发光、结果页confetti效果（可选）、按钮脉冲
- 文化语境：Y2K 千禧年互联网 / MSN / QQ空间 / 早期Web 2.0

【页面结构】（互动测试，用JS实现真实问答逻辑）

1. 封面：
   - "星球研究所 PLANETLAB" 小字
   - "你是哪种咖啡？☕" 大标题
   - "5道题，60秒，发现你的咖啡人格"
   - "开始测试 →" 按钮

2. 题目（5道，每道2-3个选项，JS控制显示/隐藏）：
   - Q1: 周一早上，你的第一反应是？
     A. 闹钟响第一下就起床（美式黑咖啡型）
     B. 再睡5分钟…好吧再5分钟（拿铁型）
     C. 直接请假（浓缩 Espresso 型）

   - Q2: 朋友约你尝试一家新餐厅，你会？
     A. 立刻答应，越新奇越好
     B. 先看看大众点评评分
     C. 我有固定去的三家店就够了

   - Q3: 你的桌面通常是什么状态？
     A. 一尘不染，所有东西都有固定位置
     B. 有点乱但我知道东西在哪
     C. 别问，问就是创意工作环境

   - Q4: 深夜12点你最可能在做什么？
     A. 已经睡了
     B. 刷手机/追剧
     C. 在做一个疯狂的 side project

   - Q5: 选一个词形容自己：
     A. 稳定
     B. 温暖
     C. 有趣

3. 结果页（根据选择最多的类型显示不同结果）：
   - A型多 → "你是 美式黑咖啡 ☕" — "简单直接，不废话。你是团队里最靠谱的那个人，效率就是你的信仰。虽然有时候被说太严肃，但你知道自己的节奏。"
   - B型多 → "你是 燕麦拿铁 🥛" — "温暖包容，人间理想型。你是朋友圈里的氛围担当，能让任何场合都变得舒服。偶尔优柔寡断，但这叫深思熟虑。"
   - C型多 → "你是 脏脏摩卡 🍫" — "不走寻常路的创意怪才。你的脑子里永远有新点子，混乱是你的燃料。别人觉得你疯狂，你觉得别人无聊。"
   - "重新测试" + "分享结果" 按钮

【技术要求】
- 单个 HTML 文件，所有 CSS 和 JS 内联
- JS实现真实的问答逻辑：点击选项记录答案，5题后计算结果，显示对应描述
- 使用 Google Fonts：Audiowide、ZCOOL QingKe HuangYou、Noto Sans SC
- 响应式设计，适配手机和桌面
- 中文为主，正文字号至少 17px
- 生产级品质
- Y2K风格：钢蓝/银/粉/紫、金属光泽边框（border gradient）、气泡装饰、星星✦、选项卡hover发光、千禧年互联网感
