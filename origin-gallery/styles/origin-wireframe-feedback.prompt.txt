请生成一个单页 HTML 网页，要求如下：

【网页类型】用户反馈评分页 / User Feedback NPS Survey / 满意度调查
【品牌】虚构产品「好用吗 HOWZIT」— 产品满意度反馈收集页
【主题】一个线框草图风格的用户反馈评分页面，像产品经理在白板上画的原型图

【视觉风格】Wireframe Sketch / 线框草图 / UX原型手绘
- 色调：白板白 (#FAFAFA)、铅笔灰 (#555555)、线框蓝 (#4A90D9)、批注红 (#E85050)、标注绿 (#4CAF50)、浅灰网格 (#E8E8E8)、高亮黄 (#FFF3CD)
- 字体：Architects Daughter（手绘建筑师字体，草图感）英文、Noto Sans SC 400/500（中文 17px+）
- 布局：原型草图式 — 白色底+CSS网格点阵(dot grid)背景，所有UI元素都用虚线border(dashed)像手绘线框，标注箭头和批注文字散布，像一张UX白板照片
- 动效：中 —
  * CSS网格点阵背景(radial-gradient小圆点)
  * 线框元素手绘感(dashed border + 不完美的border-radius)
  * JS可交互NPS评分(0-10点击选择)
  * 选择后表情变化动画
  * 红色手绘批注箭头装饰

【页面结构】

1. 头部（草图标题）：
   - "好用吗？" 大标题（Architects Daughter，铅笔灰）
   - "HOWZIT · 产品满意度调查"
   - 红色批注："← 这是标题"（手写风格小字）

2. NPS评分区（核心交互！）：
   - "你有多大可能向朋友推荐我们的产品？"
   - 0-10 数字按钮一行排列（线框虚线border，点击选中变蓝色实线）
   - 左标注"不太可能" · 右标注"非常愿意"
   - 选中后显示对应表情：0-6 😞 · 7-8 🙂 · 9-10 😍
   - JS点击交互

3. 文字反馈区（线框文本框）：
   - "告诉我们你的想法（选填）"
   - 虚线border的textarea模拟
   - 批注箭头："← 这里写感受"

4. 快速标签选择（多选toggle）：
   - "你最喜欢什么？"
   - 标签：界面设计 · 使用流畅 · 功能丰富 · 客服响应快 · 价格合理 · 文档清晰
   - 虚线border标签，选中变蓝色

5. 提交按钮：
   - "[ 提交反馈 ]"（虚线border按钮，hover变实线）
   - 批注："点这里提交 →"

6. 线框草图装饰：
   - 几条红色手绘箭头（CSS伪元素）
   - 黄色高亮标注区域
   - 蓝色标注连线

7. 页脚：
   - "好用吗 · HOWZIT · 你的反馈让产品更好"
   - "页面来源：虚构产品 · 仅供展示"
   - 批注："v2.1 · 设计稿 · 待评审"

【技术要求】
- 单个 HTML 文件，所有 CSS 和 JS 内联
- 使用 Google Fonts：Architects Daughter、Noto Sans SC
- 响应式设计，适配手机和桌面
- 中文为主，正文字号至少 17px
- 生产级品质
- 必须实现：
  1. CSS网格点阵背景(radial-gradient小圆点重复)
  2. 所有UI元素虚线dashed border（线框感）
  3. JS NPS评分0-10可点击选择（选中变色+表情变化）
  4. 快速标签多选toggle
  5. 红色手绘批注箭头（CSS伪元素）
  6. 提交按钮hover从虚线变实线
